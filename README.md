# PlayerInfo

# Command
/pinfo **&lt;player&gt;** - Shows Player name, Steam name, SteamID, GroupId, IP, Ping, Health, Hunger, Water, Virus, Stamina, Oxygen, Experience, Is Bleeding, Is Broken Legs
